# 趣味・オープンな個人開発のレポジトリ

## ディレクトリ構成

```
hobby/
  ├ README.md
  ├ config/
  │  └ {secret_key}.json
  ├ adhoc/
  │  └ {project_name}/
  │      └ {script_name}.py
  │      └ README.md
  ├ regular/
  │  └ {project_name}/
  │      └ {script_name}.py
  │      └ README.md
  ├ utils/
  |   └ {utils.py}
  |
  ├ inspection/
  └ workspace/
```

- adhoc
  - 一時的なプロジェクト
- regular
  - 定期・継続的に実行されるプロジェクト
- utils
  - 汎用性の高いコンポーネント・クラスを格納
- inspection
  - 検証用
- workspace
  - git に上がらない個人のスペース
